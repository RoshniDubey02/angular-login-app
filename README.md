# Angular Login Authentication App

This is a simple Angular application that demonstrates user login authentication with a protected dashboard.

## Features
- User login functionality
- Authentication service
- Protected dashboard route
- Logout functionality
- Responsive and clean UI
- Built using Angular and TypeScript

## Purpose
This project was created to practice Angular authentication flow, routing, and component-based architecture.

## Tech Stack
- Angular
- TypeScript
- HTML
- CSS

## How It Works
1. User logs in using the login form
2. After successful login, the user is redirected to the dashboard
3. Dashboard is accessible only to authenticated users
4. User can log out and return to the login page
