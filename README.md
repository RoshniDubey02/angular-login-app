# Angular Login Authentication App

An Angular application that demonstrates user authentication with a protected dashboard and clean routing flow.

## Features
- User login functionality
- Authentication service
- Protected dashboard route
- Logout functionality
- Responsive and clean UI
- Built with Angular and TypeScript

## Tech Stack
- Angular
- TypeScript
- HTML
- SCSS
